# Accessibility Report

Phase 9 adds a keyboard-visible skip link, a stable `main-content` landmark, global focus indicators, offline status announcements, screen-reader loading text and route-level friendly error handling.

Existing controls use labels and ARIA names across authentication, booking, search, galleries and dashboards. Reduced-motion preferences are respected globally.

Before public beta, run automated axe checks and manual keyboard tests for every route at 320 px, 768 px, 1024 px and desktop widths. Validate contrast using the final production imagery and brand assets because image overlays can change effective contrast.
