# Guest User Guide

Guests can search public listings, inspect property details and create an account to book.

## Booking journey

1. Search by destination and filters.
2. Review room, price, amenities, policies and dates.
3. Sign in or register.
4. Enter guest information and confirm checkout details.
5. Download the receipt from the booking record.

Bookings remain subject to the displayed cancellation and property policies. A guest can cancel only while the booking status permits it. Completed stays can be reviewed once; reviews should be factual and respectful.

Use Favorites and Recently Viewed to retain discovery choices. Account settings provide profile and notification controls. Report booking or account problems through Feedback or the Help Center.
