# Installation Guide

1. Install Node.js 22 LTS and npm.
2. Extract the project into a clean folder.
3. Run `npm install`.
4. Copy `.env.example` to `.env` and provide the Firebase web application values.
5. Run `npm run dev` and open the Vite URL.
6. Run `npm run check` before committing or deploying.

The project requires no source-code edits for local startup. Firebase-backed actions require a configured Firebase project and deployed rules.
