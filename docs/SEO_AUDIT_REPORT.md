- Canonical URLs derived from the configured production site URL.
- Open Graph and Twitter Card metadata.
- Organization, Website, property/lodging, breadcrumb and review structured data.

## Deployment checks

Set `VITE_SITE_URL` to the canonical custom domain before building. Submit the deployed sitemap to search consoles, validate structured data with production property records, verify social previews and ensure robots rules do not block assets required for rendering.

## Status

No source-level SEO blocker was identified. Search-engine indexing remains dependent on domain ownership, live content quality, crawl availability and operator submission.
