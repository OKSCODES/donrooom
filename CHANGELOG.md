# Changelog

All notable changes to DONROOM are documented here.

## [1.0.0] — 2026-07-18

### Commercial release
- Certified public marketplace, authentication, booking, guest, property-owner and administrator journeys.
- Added operational health checks, centralized error reporting, structured logs, maintenance mode, feature flags, feedback and Help Center.
- Added production SEO, accessibility foundations, PWA installation, offline fallback and performance monitoring.
- Added Firebase Hosting configuration, Firestore indexes, Firestore rules, Storage rules and GitHub Actions validation/deployment.
- Added complete commercial operations, security, backup, recovery, deployment and user documentation.

### Security
- Enforced role, status and ownership checks in Firestore and Storage rules.
- Preserved transactional duplicate-booking prevention and immutable audit/error records.
- Added security headers, Content Security Policy and environment validation.

### Quality
- Public npm registry lockfile.
- Route-level code splitting and production bundle optimization.
- Zero production dependency vulnerabilities at certification time.
